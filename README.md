#  Stock Price Trend Analysis System

##  Case Study Project #10 - Python Study Group

---

##  Project Overview

This project is a Stock Price Trend Analysis System built as part of the Python Study Group curriculum. It demonstrates the use of Python fundamentals (variables, data types, operators, comparisons, and logical operators) to simulate financial analysis performed by investment firms like Bamboo, Chaka, Trove, Robinhood, and others.

###  Industry Scenario

As a developer at **AlphaEdge Investments** (a FinTech company), I built this program to help financial analysts review stock performance and generate daily investment reports using Python fundamentals only.

##  Technologies Used

- **Python 3.x**
- **Jupyter Notebook** (for interactive development)
- **Variables, Data Types, Operators** (Lessons 1-5 concepts only)

---

##  Features Implemented

1. **Price Increase Calculation**
   - Computes daily price movement
   - Formula: `Current Price - Opening Price`

2. **Investment Value Calculation**
   - Calculates total portfolio value
   - Formula: `Current Price × Number of Shares`

3. **Dynamic Price Updates**
   - Updates market price using assignment operators
   - Simulates real-time market changes

4. **Performance Analysis**
   - Target price achievement check
   - Performance against expectations
   - Investment strength evaluation
   - Advisor attention requirement check

5. **Professional Report Generation**
   - Clean, formatted output
   - Ready for investment manager presentation
